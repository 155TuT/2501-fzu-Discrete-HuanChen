### t4

用等值演算法证明下列等值式

**(1)**

(p ∧ q) ∨ (p ∧ ¬q)

≡ p ∧ (q ∨ ¬q)

≡ p ∧ 1

≡ p

**(2)**

(p → q) ∧ (p → r)

≡ (¬p ∨ q) ∧ (¬p ∨ r)

≡ ¬p ∨ (q ∧ r)

≡ p → (q ∧ r)

**(4)**

(p ∨ q) ∧ ¬(p ∧ q)

≡ (p ∨ q) ∧ (¬p ∨ ¬q)

≡ [(p ∨ q) ∧ ¬p] ∨ [(p ∨ q) ∧ ¬q]

≡ [(p ∧ ¬p) ∨ (q ∧ ¬p)] ∨ [(p ∧ ¬q) ∨ (q ∧ ¬q)]

≡ 0 ∨ (q∧¬p) ∨ (p∧¬q) ∨ 0

≡(¬p ∧ q) ∨ (p ∧ ¬q)
